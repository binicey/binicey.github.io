4.数值计算方法

1.微积分计算

1.计算精度可控的数值积分

  • 函数：integral(fun,xmin,xmax)
  • 参数说明：
    • fun-用于描述被积函数的M文件函数或匿名函数，可以是标量型函数，也可以阵列型被积函数；
    • xmin-积分下限；xmax-积分上限
  • 基于自适应辛普森方法
    • [I,n ]= quad(filename,a,b,tol,trace)
  • 基于自适应 Gauss Lobatto 方法
    • [I,n]= quadl( filename,a,b,tol,trace)
  • 说明:
    • 其中filename 是被积函数名；
    • a 和 b 分别是定积分的下限和上限 积分限 [a,b] 必须是有限的 不能为无穷大 (Inf )
    • tol 用来控制积分精度(默认时取$ tol=10^{-6}$ )
    • trace 控制是否展现积分过程
      • 若取非 0 则展现积分过程
      • 取 0 则不展现 默认时取 trace= 0 ；
    • 返回参数 I 即定积分的值
    • n 为被积函数的调用次数 。
  • 基于自适应高斯 克朗罗德方法
    • [I,err]= quadgk(filename,a,b);
    • 其中err 返回近似误差范围,其他参数的含义和用法与 quad 函数相同 。
    • 积分上下限可以是无穷大 Inf 或 Inf 也可以是复数 。
    • 如果积分上下限是复数 则 quadgk 函数在复平面上求积分 。
  • 基于梯形积分法
    • I=trapz (x,y);
  • 多重积分
    • 二重积分
      • I=integral2(filename,a,b,c,d)
      • I=quad2d(filename,a,b,c,d)
      • I=dblquad(filename,a,b,c,d,tol)
    • 三重积分
      • I=integral3(filename,a,b,c,d,e,f)
      • I=triplequad(filename,a,b,c,d,e,f,tol)

2.函数极限的数值求解

  • [x,fval,exitflag,output]=fminbnd(fun,x1,x2,options)

    -求一元函数在区间x1和x2之间的极小值

  • [x,fval,exitflag,output]=fminsearch(fun,x0,options)

    -求多元函数极值点

  • 参数说明：

    • fun-待解目标函数， M文件函数或匿名函数，其中fminsearch种的多元函数应采用单一变量名的向量形式表达

    • fminbnd的第二、第三个输入x1,x2分别表示被研究区的左、右边界，输出量x,fval分别为极值点和相应的目标函数值

    • fminsearch的第二个输入量x0是搜索起点的向量或者一组搜索起点的矩阵，当采用多个搜索起点，x表示多个搜索结果

    • exitflag-若给出大于0的数，说明成功搜索到极点

      Output给出具体的优化算法和迭代次数

3.多项式

  • 多项式运算函数

    • matlab中采用按降幂排列的多项式系数组成的行向量表示多项式
      • 多项式系数向量的顺序是从高到低 。
      • 多项式系数向量包含 0 次项系数 所以其长度为多项式最高次数加 1 。
      • 如果有的项没有 系数向量相应位置应用 0 补足 。
    • 多项式创建
      • poly2sym©
      • c 为多项式的系数向量；
    • 多项式乘法
    • c=conv(a,b):a(x) ×b(x)乘积多项式的系数向量；
    • 多项式除法
      • [q,r]=deconv(b,a):b(x) /a(x) 多项式的商的系数向量q,余数的系数向量r
    • 多项式求导
      • p= polyder ( P)：求多项式 P 的导函数。
      • p= polyder (P, Q)：求 P × Q 的导函数。
      • p,q polyder (P, Q)：求 P/Q 的导函数，导函数的分子存入 p分母存入 q 。
    • 多项式求值
      • polyval (p,x )
        • 代数多项式求值 其中 p 为多项式系数向量 x 可
          是标量 、 向量或矩阵 。
        • 若 x 为标量 则求多项式在该点的值；
          若 x 为向量或矩阵 则对向量或矩阵中的每个元素求多项式值 。
      • polyvalm (p,x)
        • 矩阵多项式求值 其调用格式与 polyval 相同
          但含义不同 。
        • polyvalm 函数要求 x 为方阵 以方阵为自变量求
          多项式的值
      • 两者区别:
        • 前者为矩阵中的对应元素的代数运算;
        • 后者以x为矩阵,进行矩阵运算;
    • 部分分式分解
      • [r,p,k]=residue(b,a):因式分解b(x) /a(x) =r1/(x-p1) +r2/(x-p2)+……rn/(x-pn)+k(x)，r-为分子系数向量，p-极点系数向量，k为余数系数向量
    • 多项式求根
      • r=roots(a):多项式根
    • 根据根构建多项式
      • a=ploy®
      • 当r为一维数组，r为多项式根，a为特征多项式系数向量，当r为方阵，a表示矩阵r所对应的特征多项式的系数向量
    • Ploy2str(q,’x’)：将多项式系数向量q写成容易读的多项式

5.符号计算

1.matlab符号计算的介绍

  • MATLAB 符号计算是通过集成的符号运算工具箱（Symbolic Math Toolbox）来实现的。

  • 计算可以给出完全正确的封闭解或任意精度的数值解

  • 计算的指令比较简单，所需要的时间较长

  • 与数值运算的区别

    • 数字（数值）运算中必须先对变量赋值，然后才能参与运算。
    • 符号运算无须事先对独立变量赋值，运算结果以标准的符号形式表达。

2.符号对象的创建和使用

1.创建符号对象和表达式

  • 符号对象具有一种独立的数据类型，称为符号数据类型。
  • 单个符号对象的创建
    • sym([对象名字符串])
    • 注意对象名字符串有引号；
    • 注意符号变量无需赋予数值；
      • e.g. a=sym(′x′)a=sym('x')a=sym(′x′)
  • 多个符号变量的创建
    • syms [v1] [v2]…
    • 注意符号名之间不需要添加逗号,以空格隔开;
      • e.g.syms a b v dsyms\ a\ b\ v\ dsyms a b v d
  • 符号表达式的创建
    • 通过已定义的符号变量进行直接计算;
  • 符号矩阵的创建
    • 使用已定义的顾好变量进行举证创建;
    • 注意:输出时相比于数值举证,符号矩阵两侧有方括号;

3.符号表达式的化简

  • 合并同类项collect()
    • R=collect(S) %将表达式S中的同次幂项合并。
    • R=collect(S,v) %将表达式S中变量v的同次幂项合并。

    • R = expand(S) % 将表达式S中的各项进行展开。
  • 多项式因式分解factor()
    • R=factor(X)：如果X是一个多项式或多项式矩阵，该函数将X表示成低阶多项式相乘的形式（行向量）。
    • 注意:结果位符号行向量,每一个元素代表一个因子;

  • vpa(A,d) % 对符号计算得到的精确值进行近似，有效位数为d位，若不指定d，则按当前有效位数输出。

3.符号微积分

1.符号表达式的极限

  • limit()
    • 说明:在MATLAB中求函数极限的函数是limit，可用来求函数在指定点的极限值和左右极限值。
    • 使用格式:
      • imit(f,a)：求当默认自变量x趋近于常数a时，符号函数f(x)的极限值。当a默认时，求当默认自变量x趋近于0时的极限值。
      • limit(f,‘y’,a)：求符号函数f(y)的极限值，即计算当变量y趋近于常数a时，f(y)函数的极限值。
      • limit(f,‘y’,a,‘right’)或limit(f,‘y’,a,‘left’)：求符号函数f的极限值或。'right’表示变量x从右边趋近于a，'left’表示变量x从左边趋近于a

2.符号表达式的微分

  • diff()
    • 说明:用于对符号表达式求微分
    • 使用格式:
      • diff(s) %按findsym函数指示的默认变量对符号表达式s求一阶导数。
      • diff(s,‘v’) %以v为自变量，对符号表达式s求一阶导数。
      • diff(s,n) %按findsym函数指示的默认变量对符号表达式s求n阶导数，n为正整数。
      • diff(s,‘v’,n) %以v为自变量，对符号表达式s求n阶导数。

3.符号表达式的积分

  • int()
    • 说明:求符号积分;
    • 使用格式:
      • int(s)：没有指定积分变量和积分阶数时，系统按findsym函数指示的默认变量对被积函数或符号表达式s求不定积分。
      • int(s,v)：以v为自变量，对被积函数或符号表达式s求不定积分。
      • int(s,v,a,b)：求定积分运算。a、b分别表示定积分的下限和上限。
        • 该函数求被积函数在区间[a，b]上的定积分。
        • a和b可以是两个具体的数，也可以是一个符号表达式，还可以是无穷（inf）。
        • 当函数f关于变量x在闭区间[a，b]上可积时，函数返回一个定积分结果。
        • 当a、b中有一个是inf时，函数返回一个广义积分。
        • 当a、b中有一个符号表达式时，函数返回一个符号函数。

4.级数求和

  • symsum()
    • 说明:求以符号表达式为通项的级数;
    • 使用格式:
      • symsum(s,v,n,m)
        • s表示级数的通项（符号表达式）；
        • v是求和变量，v省略时使用系统的默认变量；
        • n和m是求和的开始项和末项。

5.泰勒级数

  • taylor()
    • 说明:将一个任意函数表示为一个幂级数。
    • 使用格式:
      • taylor(f,v,a,name,value)
        • taylor函数将函数f按变量v展开为泰勒级数，展开到第n项（即变量v的n −1次幂）为止，n的默认值为6。
        • 参数a指定将函数f在自变量v = a处展开，a的默认值是0。

4.符号方程求解

1.代数方程

  • solve()
    • 说明:求解用符号表达式表示的代数方程;
    • 使用格式:
      • solve(s)
        • 求解符号表达式s的代数方程，变量为默认变量。
      • solve(s,v)
        • 求解符号表达式s的代数方程，求解变量为v。
      • solve(s1,s2, …,sn,v1,v2,…,vn)
        • 求解符号表达式s1，s2，…，sn组成的代数方程组，求解变量分别为v1，v2，…，vn。
    • 注意:
      • solve函数能求解一般的线性、非线性或超越代数方程。
      • 对于不存在符号解的代数方程组，若方程组中不包含符号参数，则solve函数给出方程组的数值解。
  • 求解步骤
    • 先定义符号变量，再创建符号表达式
    • 利用字符“==”创建方程表达式

2.微分方程

  • dslove()
    • 说明:在MATLAB中，用大写字母D表示导数。例如，Dy表示y’，D2y表示y’’，Dy(0)=4表示y’(0)=4。微分初始条件写成y(a)=b，Dy©=d的格式。
    • 使用格式:
      • dsolve(e,c,v)。
        • 该函数求解常微分方程e在初值条件c下的特解。
        • 参数v描述方程中的自变量，省略时按默认原则处理，若没有给出初值条件c，则求方程的通解。
      • dsolve(e1,e2, …,en,c1, …,cn,v1, …,vn)
        • 该函数求解常微分方程组e1，…，en在初值条件c1，…，cn下的特解，若不给出初值条件，则求方程组的通解，v1，…，vn给出求解变量。
        • 若边界条件少于方程（组）的阶数，则返回的结果中会出现任意常数C1，C2，…。dsolve命令最多可以接受12个输入参量（包括方程组与定解条件个数）。
        • 若没有给定输出参量，则在命令窗口显示解列表。若该命令得不到解析解，则返回一警告信息，同时返回一空的sym对象。这时，用户可以用命令ode23或ode45求解方程组的数值解。

    • tlims 为参数函数自变量的取值范围,用二元向
      量 [tmin,tmax] 描述 默认为 [5 ,5] 。
    • 可以指定曲线的线型 、 颜色或数据点标记.

5.三维曲面

1.平面网络的生成

  • 定义:用矩阵X、Y分别存储每一个小矩形顶点的x坐标与y坐标，矩阵X、Y就是该矩形区域的xy平面网格坐标矩阵。
    • 网格矩阵表示了曲面点在xy平面的分布.
  • 生成方法
    • 利用矩阵运算生成
    • 利用 meshgrid 函数生成
      • [X,Y]=meshgrid(x,y)
      • 参数说明:
        • 参数 x 、 y 为向量 存储网格点坐标的 X 、 Y 为矩阵 。

2.三维曲面的绘制

  • mesh(x,y, z, c)(网图)
  • surf(x,y, z, c)（曲面图）
  • 参数说明:
    • x 、 y 是网格坐标矩阵
    • z 是网格点上的高度矩阵
    • c 用于指定在不同高度下的曲面颜色 。 c 省略时 颜色的设定正比于图形的高度.
  • mesh(z,c)
  • surf(z,c)
  • 参数说明:
    • 当x 、 y 省略时, z 矩阵的第 2 维下标当作 x 轴坐标 ,z 矩阵的第 1 维下标当作 y 轴坐标 。
  • 其他功能函数:
    • 带等高线的三维网格曲面函数 meshc
    • 带底座的三维网格曲面函数 meshz
    • 具有等高线的曲面函数 surfc
    • 具有光照效果的曲面函数 surfl

3.标准三维曲面

  • sphere 函数(球面)
    • [x,y,z ]=sphere(n)
    • 参数说明:
      • 产生3 个 ( n+1) 阶的方阵 采用这 3 个矩阵可以绘制出圆心位于原点 、 半径为 1 的单位球体 调用时不带输出参数 直接制球面 n 决定球面圆滑程度 (默认 20)
  • cylinder 函数 (柱面)
    • [x,y,z ]=cylinder(n)
    • 参数说明:
      • 其中参数 R 是一个向量,存放柱面各个等间隔高度上的半径,n 表示在圆柱圆周上有 n 个间隔点,默认有 20 个间隔点,如果 R时标量,生成圆柱面

4.参数方程绘制曲面

  • fsurf(funx ,uny,funz,uvlims)
  • fmesh(funx ,uny,funz,uvlims)
  • 参数说明:
    • funx 、 funy 、 funz 代表定义曲面 x 、 y 、 z 坐标的函数,通常采用函数句柄的形式 。
    • uvlims 为 funx 、 funy 和 funz 的自变量的取值范围
    • 用 4 元向量[ umin, umax, vmin, vmax] 描述 默认为 [-5,5,-5,5] 。

6.图形修饰处理

1.视点处理

  • 基本定义:

    • 方位角：视点与原点连线在xy 平面上的投影与 y 轴负方向形成的角度,正值表示逆时针,负值表示顺时针 。
    • 仰角：视点与原点连线与xy 平面的夹角,正值表示视点在 xy 平面上方, 负值表示视点在 xy 平面下方 。

  • view(az,el)

  • 参数说明:

    • az 为方位角
    • el 为仰角

  • view 函数的其他用法

    • view(x,y,z);
    • view(2);
    • view(3);

2.色彩处理

  • 颜色的向量表示

    • [R G B]
      • 向量元素在[0,1]范围内取值.

  • 色图

    • 内建色图

    • 指定当前图形使用的色图

      • colormap(cmap)

    • 创建色图矩阵

      • 色图矩阵的每一行是RGB 三元组。可以自定义色图矩阵，也可以调用 MATLAB 提供的函数来定义色图矩阵。

  • 三维图形表面的着色

    • shading 命令

      参数效果
      shading faceted每个网格片用其高度对应的颜色进行着色网格线是黑色 。 这是默认着色方式
      shading flat每个网格片用同一个颜色进行着色 且网格线也用相应的颜色
      shading interp网格片内采用颜色插值处理 。

    • hidde on（off）：对于网图而言，on为透明网图；off为非透明网图；

  • 图形的裁剪处理

    • 将图形中需要裁剪部分对应的函数值设置成NaN ，这样在绘制图形时，函数值为 NaN 的部分将不显示出来，从而达到对图形进行裁剪的目的。
